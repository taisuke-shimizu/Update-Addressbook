# Update-Addressbook

`Update-Addressbook.ps1` は、既存のアドレス帳CSVを最新のメールアドレス情報（取得者リスト・メーリングリスト）で更新し、新しいアドレス帳ファイルを生成する PowerShell スクリプトです。

## 機能概要

既存のアドレス帳 (`address.csv`) と、desknet's NEO から取得した最新の2つのリストを比較・マージします。

- **新規追加**: 新しいリストにのみ存在するメールアドレスを新規レコードとして追加します。
- **削除・退職判定**: 既存のアドレス帳に存在し、新しいリストに存在しない社内アドレス（`@fmget.co.jp`）について、無効になったと判定し氏名の末尾に `(退職済)` または `(削除済)` を付与します。
- **重複排除**: アドレス帳内の重複レコードを整理します。
- **エンコーディング**: 日本語環境のメールシステムでそのまま読み込めるよう、入出力ともに **Shift-JIS** 形式を使用します。

### 処理パターン

| `address.csv` | 最新リスト (個人/ML) | 処理内容 |
| :---: | :---: | :--- |
| 有 | 有 | 変更なし |
| 有 | 無 | 氏名の末尾に `(退職済)` または `(削除済)` を付与 |
| 無 | 有 | 新規レコードとして追加 |

> **Note:** 社外アドレス（`@fmget.co.jp` 以外）については削除判定の対象外となり、そのまま保持されます。

## 動作環境

- **PowerShell 7** 以降を推奨

## 使い方

1. スクリプト (`Update-Addressbook.ps1`) と同じフォルダに、以下の3つの CSV ファイルを配置します。
    - **`address.csv`**
        - ベースとなるアドレス帳ファイル（Shift-JIS形式）
        - Webメール等からエクスポートしたもの
    - **`メールアドレス取得者（all）最新.csv`**
        - メールアドレス更新対象の個人リスト
        - desknet's NEO からダウンロード
    - **`メーリングアドレス(all).csv`**
        - メーリングリストの参加者リスト
        - desknet's NEO からダウンロード

2. スクリプトを実行します。

   **コマンドラインから実行する場合 (PowerShell 7)**:
   ```powershell
   pwsh ./Update-Addressbook.ps1
   ```

   **エクスプローラーから実行する場合**:
   スクリプトファイルを右クリックし、「プログラムから開く」>「pwsh.exe」を選択して実行します。

3. 処理が完了すると、同じフォルダに **`new-address.csv`**（Shift-JIS形式）が出力されます。

## 入出力ファイル詳細

| ファイル名 | 役割 | 取得元・備考 |
| :--- | :--- | :--- |
| `address.csv` | ベースファイル (入力) | Webメールからエクスポート |
| `メールアドレス取得者（all）最新.csv` | 最新個人リスト (入力) | desknet's NEO 文書管理 (ID: 33467) |
| `メーリングアドレス(all).csv` | 最新MLリスト (入力) | desknet's NEO 文書管理 (ID: 33466) |
| `new-address.csv` | 更新結果 (出力) | スクリプトによって生成されるファイル |
